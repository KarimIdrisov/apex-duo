# Apex Duo — визуальный стиль (направление «Hollywood Animal → люкс-F1»)

Дизайн-документ. Цель: перенести **визуальную иерархию и «плёночное» ощущение
Hollywood Animal** (золото-на-тёмном «люкс», портреты-карточки, ар-деко рамки и
типографика, мягкий тёплый грейд) на современный контекст **F1 2026**, не
скатываясь в буквальный ретро-пастиш 1930-х.

Статус: **план** (решение 2026-06-10: вариант «люкс-иерархия под F1», без полной
сепии). Реализация — поэтапно после согласования. Этот документ — единый источник
правды по визуалу; держать в синхроне с `main.gd`/темой при изменениях.

---

## 1. Что берём у Hollywood Animal, а что нет

Из исследования арт-направления HA (источники — см. researcher findings):
изометрический «диорамный» мир Золотого века, **ар-деко** геометрия, мягкая
сепия + тёплая десатурация, **золото только как акцент** (выделение, заголовки,
ключевые цифры) на тёмных «деревянных»/угольных фонах, кремовый текст, портреты-
карточки с многоуровневой детализацией, плёночный пост (виньетка + зерно + грейд),
капс-заголовки ар-деко + чистый гротеск для данных.

| Берём (структура/ощущение)                          | Адаптируем под F1 / не берём          |
|------------------------------------------------------|----------------------------------------|
| Иерархия **золото-на-тёмном** (золото = акцент)       | Тёплую сепию-палитру → **карбон-чёрный** база |
| **Портреты-карточки** гонщиков/персонала, золотая рамка | Полную ар-деко сепию мира (у нас 2D-HUD) |
| **Плёночный пост-слой** (виньетка + лёгкое зерно + грейд) | Сильную сепию → лишь **лёгкий тёплый грейд** |
| **Ар-деко акценты**: тонкие золотые линейки, капс-заголовки, симметрия | Деревянные текстуры, ретро-орнаменты |
| Минимализм «больше стиля, чем зрелища»               | Перегруженность рамками везде           |
| Капс + трекинг в заголовках, чистый гротеск в данных | Декоративные засечки в теле текста      |

**Главный принцип читаемости как «люкс, а не просто жёлтый»:** золото — только
на акцентах (выделенное состояние, заголовки, ключевые числа, тонкие линейки) на
тёмных фонах; основной читаемый текст — кремовый/светлый.

### 2.6 Цвета шин (гоночная конвенция — менять осторожно)

Шины — это **реальная F1-конвенция** (soft=red, medium=yellow, hard=white,
inter=green, wet=blue). Менять не рекомендую: игроки читают их мгновенно.
Оставляем как есть, при необходимости лишь чуть теплее под грейд. Текущие:
soft `#e23b3b`, medium `#f2c14e`, hard `#e8e8e8`, inter `#3fb950`, wet `#3b82f6`.

## 3. Типографика

Сейчас: дефолтный шрифт Godot, размеры 13–52, без кастомных гарнитур.

**План — две гарнитуры (две роли):**

1. **Дисплей/заголовки — ар-деко геометрик, капс + трекинг.** Бесплатные,
   period-true и хорошо читаемые: **Jost** (геометрик Futura-like), **Cinzel**
   (деко-серив для «премиум» титулов), **Bebas Neue** (высокий конденс-капс для
   крупных чисел/титула «APEX DUO»). Рекомендация: **Jost** для заголовков
   панелей + **Bebas Neue** для главного титула и крупных позиций на табло.
2. **Тело/данные — чистый гуманистический гротеск.** **Jost** (тот же,
   обычное начертание) или **Archivo/Work Sans**. Один гротеск на всё тело — меньше
   риск рассинхрона.

**Кириллица обязательна** (вся игра на русском). Проверить покрытие кириллицы:
Jost — есть; Bebas Neue — латиница, кириллицу надо брать из **Oswald** (близкий
конденс-капс с кириллицей) как замену для главного титула. **Решение по титулу:
Oswald (капс) вместо Bebas Neue ради кириллицы.**

**Конвенции:**
- Заголовки панелей — **КАПС**, letter-spacing, золото `#D4AF37`, под ними тонкая
  золотая линейка (1px `#8C6B2F`).
- Тело — кремовый `#F3E9D2`, обычный кейс.
- Крупные числа (позиции, очки, время) — дисплейная гарнитура, золото на ключевых.

**Технически в Godot:** положить `.ttf` в `res://fonts/`, собрать `Theme`-ресурс
или назначать `FontFile` через `add_theme_font_override("font", ...)` в хелперах
`_mklabel`/`_label`/`_cell`. Лицензии шрифтов (SIL OFL) — закоммитить рядом.

## 4. Компоненты UI (StyleBox / рамки)

Сейчас: плоские `StyleBoxFlat`, скругление 10px, без границ/теней (кроме теней
машин на миникарте). Цель — «люкс»-рамки в духе деко, но сдержанно.

**Базовая панель (`_panel_container`) — новый стиль:**
- `bg_color = PANEL (#1A1712)`
- **Скругление 4px** (деко — более «острые» углы, не 10px). Решить: 4 vs 6.
- Тонкая **бронзовая граница** 1px `#2C2618` (вместо «безрамочности»).
- content margin 14px (оставить).
- Опционально: верхняя золотая линейка-акцент (отдельный `ColorRect`/`draw`),
  как «карниз» деко-панели — только на ключевых панелях (табло, паддок-хедер).

**Выделенное состояние (карточки команд/гонщиков):**
- Сейчас красная рамка 3px `#c8102e` → **золотая рамка 2px `#D4AF37`** +
  внешний бронзовый контур `#5E4A20` (двойная линия = деко).

**Кнопки:**
- Обычная: фон `#221E17`, текст кремовый, бронзовая граница 1px.
- Hover: граница → золото `#D4AF37`, текст → шампань `#E4D48B`.
- Pressed/выбрано: фон чуть светлее `#2A2418`, граница золото 2px.

**Прогресс-бары (заряд/износ/настроение):** заливка — семантический цвет (§2.5),
фон-жёлоб `#221E17`, тонкая бронзовая обводка. Высоту оставить 14–16px.

## 5. Портреты-карточки гонщиков/персонала (ключевой паттерн HA)

Самый узнаваемый и переносимый элемент Hollywood Animal.

**Паттерн:**
- Карточка = `PanelContainer` (тёмный фон) → сверху **портрет** (PNG/процедурный),
  обёрнутый в **золотую деко-рамку** (двойная линия золото+бронза, углы со
  «ступенькой»), под портретом — **капс-имя** на тонкой золотой линейке, ниже —
  атрибуты/мораль/форма.
- **Многоуровневая детализация (как в HA):** ключевые гонщики (наши 2 + соперники-
  звёзды) — богатая рамка; рядовой персонал (`personnel.gd`) — упрощённая рамка.
- **Где применить:** `season_setup.gd` (выбор команды — уже карточки, добавить
  портрет+рамку), `season_hub.gd` (паддок: карточки гонщиков с моралью/развитием),
  пред-гоночная модалка (состав).

**Источник портретов:** на этапе прототипа — **процедурные плейсхолдеры**
(инициалы/силуэт в золотой рамке + командный цвет), позже — PNG-ассеты. Рамку и
капс-имя делаем сразу, портрет — заглушка. Это даёт «HA-вид» без арт-пайплайна.

## 6. Плёночный пост-обработка слой (атмосфера)

Даёт «старое студийное» ощущение HA без вмешательства в gameplay/детерминизм —
полностью UI-сторона.

**Архитектура:** верхний `CanvasLayer` (layer высокий) → полноэкранный `ColorRect`
с шейдером. Полностью развязан с симом (нет риска для детерминизма netcode).

**Шейдер (canvas_item) делает:**
1. **Тёплый грейд** — лёгкий сдвиг в тёплые тона (НЕ полная сепия): множитель
   ~`vec3(1.06, 1.0, 0.92)` + малое поднятие средних. Сила настраиваемая (uniform
   `warmth`, дефолт ~0.15).
2. **Виньетка** — мягкое затемнение по краям (uniform `vignette_strength` ~0.35,
   `vignette_softness`).
3. **Зерно плёнки** — тонкий анимированный шум (uniform `grain_amount` ~0.04,
   движется по `TIME`). Лёгкое — чтобы не «грязнило» текст.
4. (Опц.) **Очень лёгкая хром-аберрация** по краям — обычно перебор для UI,
   по умолчанию выключить.

**Важно про рендерер:** проект на `gl_compatibility`. Чтение `SCREEN_TEXTURE`
(`hint_screen_texture`) в canvas-шейдере в Compatibility-режиме Godot 4.x
**работает**; HDR-блюр «из коробки» — нет, но он нам и не нужен. Грейд/виньетка/
зерно — дёшевы. Проверить на этапе реализации headless’ом/в редакторе.

**Переходы экранов:** медленный fade-to-black или «ирисовый» круг (кинолента) —
дешёвый `ColorRect` + tween поверх. Опционально, этап полировки.

**Все эффекты — с тумблером** (для слабых машин / читаемости): `uniform`-силы +
глобальный флаг в настройках.

## 8. Рефакторинг: где живёт стиль (важно сделать ПЕРВЫМ)

Сейчас цвета **захардкожены и продублированы** в `main.gd`, `season_setup.gd`,
`season_hub.gd`, плюс инлайн в `track_map.gd`. Менять палитру = править 3+ файла.

**План централизации (предусловие любого визуального изменения):**
1. Создать **`theme.gd` (class_name Palette)** — один `const`-словарь всех цветов
   + размеров шрифтов + хелперы (`panel_stylebox()`, `gold_border()`,
   `heading_label()`, `body_label()`, `card_frame()`). Согласуется с конвенцией
   проекта «туниблы — `const`-словари сверху файла».
2. `main.gd`, `season_setup.gd`, `season_hub.gd`, `track_map.gd` — заменить
   локальные `const ACCENT/BG/PANEL/...` и инлайн-`Color("...")` на `Palette.*`.
3. Хелперы `_mklabel/_label/_cell/_button/_panel_container` — переписать поверх
   `Palette` (шрифт + цвет + stylebox в одном месте).
4. (Опц.) Собрать `.tres` `Theme` для дефолтных оверрайдов, чтобы меньше
   ручных `add_theme_*` на каждый виджет.

Без этого шага любая дальнейшая визуальная работа размазывается по файлам.

## 9. Поэтапный roadmap

Порядок по принципу «максимум вида за минимум риска», каждый этап — отдельный,
проверяемый, не ломающий сим/неткод.

- **Этап 0 — Централизация (`theme.gd`).** Вынести палитру/шрифты/хелперы. Чисто
  рефакторинг, визуал не меняется. *Предусловие всего остального.*
- **Этап 1 — Палитра.** Переключить `Palette` на карбон+золото (§2). Мгновенно
  даёт «люкс-тёмный» вид по всем экранам.
- **Этап 2 — Типографика.** Подключить `Jost` + `Oswald`(капс), капс-заголовки с
  золотой линейкой, кремовый текст. (Проверить кириллицу.)
- **Этап 3 — Рамки/компоненты.** Деко-StyleBox: бронзовые границы, золотое
  выделение, кнопки hover/pressed. Решить P6-цвет и скругление 4 vs 6.
- **Этап 4 — Плёночный пост-слой.** `CanvasLayer`+шейдер (грейд/виньетка/зерно) с
  тумблерами. Проверить в Compatibility-рендерере.
- **Этап 5 — Портреты-карточки.** Деко-рамка + капс-имя + плейсхолдер-портрет в
  `season_setup`/`season_hub`/пред-гонке.
- **Этап 6 — Миникарта под стиль** + переходы экранов (полировка).

Каждый этап: реализовать → линт (fresh-file трюк) → проверка в реальном движке
(godot MCP, если подключён) → обновить README/этот документ.

## 10. Открытые решения (нужно подтвердить перед реализацией)

1. **Цвет P6**: «второе золото/шампань» `#C9B07E` (моно-люкс, но машины
   труднее различать) **vs** приглушённый тил `#5BB8C4` (контраст, читаемость в
   гонке). Рекомендация: **тил** ради читаемости пары машин.
2. **Скругление панелей**: 4px (острее, деко) vs 6px (мягче). Рекомендация: **6px**
   как компромисс с современным F1.
3. **Сила грейда**: насколько тёплый. Рекомендация: лёгкий (`warmth ~0.12`), без
   ухода в жёлтый.
4. **Шрифт главного титула**: Oswald (капс, кириллица) — подтвердить вкус.
5. **Красный акцент**: оставить яркий Ferrari `#c8102e` или приглушить до
   деко-вина `#9E2B25`. Рекомендация: **приглушить**, ярко-красный спорит с золотом.
