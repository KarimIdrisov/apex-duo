# Apex Duo — визуальный стиль (направление «строгий формальный люкс F1»)

Дизайн-документ. Финальное направление (решение 2026-06-10): **строгий,
формальный, почти монохромный «табло-люкс»** — нейтральный чёрный фон, кремовый/
серый текст, **золото как точечный акцент**, острые прямые углы, тонкие hairline-
линии, плоские прямоугольные индикаторы. Ориентир — официальное хронометражное
табло F1 / деловой терминал, а **не** ар-деко опулентность Hollywood Animal.

Из Hollywood Animal взято только **структурное**: иерархия «акцент-на-тёмном»,
портреты-карточки персонала, сдержанность «больше стиля, чем зрелища». Тёплая
сепия, деко-рамки, орнаменты, свечения — **отброшены** как «мультяшные» для
современного F1 (итерация макетов 2026-06-10).

Макеты выбранного варианта: `docs/mockups/` (меню, гоночный HUD, паддок).
Этот документ — единый источник правды по визуалу; держать в синхроне с `theme.gd`.

---

## 1. Принципы

1. **Монохром-первый.** Базу несут кремовый текст и серые оттенки на нейтральном
   чёрном. Цвет — только там, где он функционален (статусы, команды игрока, шины).
2. **Золото — точечно.** `#C9A227` только на акцентах: короткий маркер у
   заголовка, выбранная строка, ключевая цифра, стрелка действия. Никаких золотых
   заливок/рамок «везде».
3. **Острая геометрия.** Скругление 2px (почти прямые углы). Никаких «пилюль».
4. **Hairline-система.** Разделители и обводки — 1px тонкие линии, не толстые.
5. **Без декора-шума.** Нет свечений машин, нет зерна, виньетка едва заметна.
6. **Плотность как у табло.** Данные читаются как хронометраж: выровненные сетки,
   капс-подписи, контраст весом шрифта, а не цветом.

## 3. Типографика

Сейчас: дефолтный шрифт Godot, размеры 13–52, без кастомных гарнитур.

**План — две гарнитуры (формальные):**
1. **Дисплей/заголовки/числа — конденс-капс гротеск.** **Oswald** (есть кириллица —
   замена Bebas ради русского). Капс, умеренный трекинг (`letter-spacing` 1.5–3),
   контраст **весом**, а не цветом. Для главного титула — крупный Oswald капс.
2. **Тело/данные — чистый гуманистический гротеск.** **Jost** (или Archivo/Work
   Sans). Обычный кейс, кремовый цвет.

Кириллица обязательна (вся игра на русском) — Oswald и Jost её покрывают.

**Конвенции:**
- Заголовки панелей — КАПС Oswald, кремовый, перед ними **короткий золотой маркер**
  (прямоугольник 3×13px), под ними **hairline-линия** на всю ширину (не золотая).
- Подписи столбцов/полей — мелкий fine `#5C5A54`, капс.
- Крупные числа (время круга, очки, позиции) — Oswald bold, кремовый; золото только
  на действительно ключевой цифре (лидер, уровень R&D выбранного).

**Технически:** `.ttf` в `res://fonts/` (Oswald, Jost; лицензии SIL OFL рядом),
назначать через `Theme`/`add_theme_font_override` в хелперах `Palette`.

## 4. Компоненты UI (строгие)

**Базовая панель (`Palette.panel`):**
- `bg_color = PANEL (#141416)`, **скругление 2px**, граница 1px `#28282C`. Без
  градиентов/теней.

**Заголовок секции (`Palette.heading`):** короткий золотой маркер 3×13px + КАПС
Oswald кремовый (ls≈2.5) + hairline-линия `#202023` под ним на ширину панели.

**Строка-выбор (таблицы/списки):** фон `#1A1A1C` + **тонкая боковая полоска 3px**
командного/золотого цвета. Без рамок-свечений.

**Кнопки (`Palette.btn`):** прямоугольник 2px, фон `#1A1A1C`, обводка 1px цветом
подписи (обычно `MUTED`), текст КАПС. Первичная (напр. «На гонку») — тёмно-винная
подложка `#1E1715` + обводка/текст акцента. Hover: обводка → золото; pressed: фон
чуть светлее. Никаких ярких заливок.

**Индикаторы/бары (`Palette.bar`):** **плоский прямоугольник** (rx=0), жёлоб
`#0F0F10` + hairline-обводка, заливка десатурированным семантическим цветом на
~90% непрозрачности. Высота 11–14px. Числовое значение справа, серым.

**Чипы/теги:** прямоугольные, тонкая обводка цветом, без заливки.

## 5. Портреты-карточки гонщиков/персонала (строгий вид)

Паттерн HA, но формально:
- Карточка = панель + тонкая боковая полоска команды. Слева **портрет** в **одной
  тонкой рамке 1px** (без двойных деко-линий и угловых точек), с **одной сдержанной
  золотой линией снизу** как «подпись».
- Портрет на этапе прототипа — **нейтрально-серый силуэт** (голова+плечи) + крупные
  инициалы. Никакой раскраски/свечения. Позже — PNG-арты (тогда тоже сдержанные).
- Справа: тег P5/P6 + №, капс-имя на hairline-линии, национальность (серым),
  атрибуты плоскими серыми барами с числом, строка «Настроение».
- Многоуровневость HA сохраняем: ключевые гонщики — карточка; рядовой персонал
  (`personnel.gd`) — компактный вариант той же сетки.
- Где: `season_setup.gd` (выбор), `season_hub.gd` (паддок), пред-гонка (состав).

## 6. Плёночный пост-обработка слой (минимальный)

Для строгого варианта — **сильно сдержанный**: только лёгкая виньетка, опционально
едва заметное зерно (по умолчанию выкл). Тёплого сепия-грейда **нет** (он и делал
вид «ретро»). Цель — чуть «утопить» края экрана, без атмосферного фильтра.

**Архитектура:** верхний `CanvasLayer` + полноэкранный `ColorRect` с шейдером,
полностью развязан с симом (нет риска детерминизму netcode). Uniform-силы +
глобальный тумблер (читаемость/слабые машины).

**Рендерер:** проект на `gl_compatibility`. Чтение `SCREEN_TEXTURE`
(`hint_screen_texture`) в canvas-шейдере в Compatibility Godot 4.x работает;
виньетка/зерно дёшевы. Проверить в редакторе/headless на этапе реализации.

Переходы экранов: простой fade-to-black (`ColorRect`+tween). Этап полировки.

## 7. Миникарта (`track_map.gd`) — РЕАЛИЗОВАНО (этап 3)

- Трасса — приглушённый тёплый серый (`#34322C` тармак, `#4A453A` racing line),
  нейтральный тёмный infield `#121110`, без ярких бордюров-акцентов сверх реальной
  красно-белой конвенции kerb (kerb сохранён как F1-конвенция).
- Машины — **сохранён информативный F1-силуэт** (отклонение от первоначального
  «точки»: аркадно читались именно свечения/трейлы, а не сам силуэт). **Убраны:
  свечения-ореолы, трейлы (вся система `trails`/`TRAIL_LEN`), оранжевый
  attack-нимб `#ff7a1a`.** Командные машины помечены **тонким кольцом 1.6px**
  (P5 золото / P6 сине-стальной); номера позиций — капс Oswald.
- Зона обгона — сдержанная `Palette.INFO` (α≈0.30); рекуперация/пит-блипы —
  `Palette.GOOD`; SC-оверлей — приглушённый янтарь.
- Заголовок трассы — капс Oswald + cream (этап 2).
- Цвета машин-соперников остаются их реальными ливреями (информативно, не chrome).

## 8. Рефакторинг: где живёт стиль (делаем ПЕРВЫМ — Этап 0)

Сейчас цвета захардкожены и продублированы: `main.gd:20-24`,
`season_setup.gd:8-10`, `season_hub.gd:13-18`, плюс инлайн в `track_map.gd`,
`_tire_color/_wear_color/_soc_color` (`main.gd:830-866`), event feed
(`main.gd:568-581`).

**`theme.gd` (class_name `Palette`)** — единый модуль:
- `const` все цвета §2 + размеры шрифтов + (позже) `FontFile` Oswald/Jost.
- Хелперы: `panel() -> StyleBoxFlat`, `heading(parent, text)`,
  `bar_styleboxes()`, `btn(text, col, primary)`, `body_label/heading_label`,
  `tire_color(c)`, `wear_color/soc_color`.
- Все экраны и `track_map` ссылаются на `Palette.*`, локальные `const ACCENT/BG/...`
  удаляются.

- **Этап 2 — Типографика.** Подключить Oswald(капс)+Jost, капс-заголовки с золотым
