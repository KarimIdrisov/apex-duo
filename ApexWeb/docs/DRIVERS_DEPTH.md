# Углубление пилотов (G1–G4)

Статус: реализовано (2026-06). Схема сейва v23. Делает вкладку «Пилоты» живой и
управляемой. Всё детерминированно; влияние на сим — через `overall`/атрибуты и `moraleMod`.

## G1 — статистика сезона + дуэль с напарником
У каждого пилота `stats` за сезон: победы, подиумы, поулы (старт P1), очки, сходы,
лучший финиш, и H2H — `qH2H` (квала) и `rH2H` (гонка) против напарника. Обновляется в
`applyResult` из финиша + стартовых позиций (`raceInfo.starts` из `main`). Сбрасывается в
межсезонье (`developDrivers`). В карточке — плашка статистики и строка дуэли.

## G2 — тренировка пилота
`driver.training` — фокус из `TRAINING` (квала / резина / гонка-обгон / дождь /
стабильность). Выбранные атрибуты растут быстрее: `TRAIN_RACE` за гонку (видимый прогресс
по ходу сезона) + `TRAIN_WINTER` зимой (`developDrivers`). В карточке — селектор тренировки
и **стрелки развития** у каждого атрибута (▲/▼ по возрастной кривой `attrDrift` + фокусу).

## G3 — настроение и запросы
Мораль (`tickDriverRace`) теперь зависит от финиша-vs-ожидания, дуэли с напарником и
короткой **формы** (EMA результатов); причина читается в карточке (`moraleReason`). Иногда
пилот выставляет **запрос** (`makeDriverRequest`): уверенно бьёт напарника → хочет статус
первого номера; контракт кончается + в форме → хочет продление. Игрок принимает/отклоняет
(`resolveDriverRequest`): принять контракт — плата + мораль↑; принять «1-й номер» — статус
lead, напарник → support (его мораль↓); отклонить — мораль↓.

## G4 — развитие черт
В межсезонье пилот, у которого ключевой атрибут перешёл порог мастерства (`TRAIT_GATE`
0.90), открывает соответствующую черту (`maybeGainTrait`, одна за сезон) + новость.
Молодой после сильных квал становится «Квалифайером» и т.п. — идентичность растёт.

## Карточка (UI)
Богатая карточка вместо текста: аватар + имя + статус (★1-й/саппорт) + OVR; форма и
настроение барами с причиной; контракт + «Продлить»; панель запроса (если есть); плашка
статистики; дуэль с напарником; селектор тренировки; полный бар-чарт 13 атрибутов со
стрелками развития.

## Проверено
Логика — самодостаточным node-прогоном: тренировка подняла квалу 0.88→0.897 (по сезону +
зима), overall +0.014, форма 0.79, мораль 0.81, пилот запросил «1-й номер»; черта ждёт
перехода атрибута через 0.90. UI собран по тем же паттернам, что рабочие карточки «Машины».

## Бэклог
H2H можно показать визуальным баром; статус «1-й номер» — завести в командную стратегию
(приоритет питов/обгонов в симе); запросы — расширить (хочет апгрейд, недоволен машиной).
